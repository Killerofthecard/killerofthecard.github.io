一个token的视角｜重新看Self-Attention机制

作为大模型的基础，本文来系统梳理一下Self-Attention的原理，并从token的视角来讲解其中发生的事情。

1. 矩阵形式

许多blog或教程总是从token的视角来切入Self-Attention机制的讲解，笔者认为这不够直观，对于初学者而言从token向量来看会引入过多的下标，反而增加了理解的难度。我们考虑矩阵形式引入，问题就会变得清晰、自然：
$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V,$$
其中：

  • $X = [x_1, x_2, \dots, x_n] \in \mathbb{R}^{n \times d_{\text{model}}}$ 为输入序列的n个token，经过embedding层后每个token被映射成一个 $d_{\text{model}}$维的向量，也就是每一行都代表一个token向量。
  • $Q = XW_Q \in \mathbb{R}^{n \times d_k}$
  • $K = XW_K \in \mathbb{R}^{n \times d_k}$
  • $V = XW_V \in \mathbb{R}^{n \times d_v}$
    注意到这里三个映射矩阵维度为：$W_Q \in \mathbb{R}^{d_{\text{model}} \times d_k}$，$W_K\in\mathbb{R}^{d_{\text{model}} \times d_k}$， $W_V \in\mathbb{R}^{d_{\text{model}} \times d_v}$，这里的 $d_k$和 $d_v$可以不同（在标准Transformer论文中设置成相同可能是工程上简化的考虑，实际上此处的差异又可以划分出不同的attention机制，我们下回分解）。

我们分开看，矩阵Q本质是一个n行$d_k$列的矩阵，每一行象征着一个token在查询（query）空间中的映射， $K^T$本质是一个$d_k$行n列的矩阵，每一列象征着一个token在键（key）空间中的映射，二者做矩阵乘法时，本质上是利用矩阵Q的行向量与矩阵 $K^T$的列向量做点积。而在向量空间中点积其实等价于余弦相似度（二者只差了一个标量分母），这就意味着此处矩阵相乘的含义是计算两两token之间（严格来说是key和value空间中的映射）的相似性。

理解了这一点，剩下的就比较显然了。 $QK^T$的结果是一个n行n列的矩阵，其中（i，j）元素表示第i个token和第j个token之间的相似性分数，分数越高二者越相似。将结果经过缩放以及softmax处理（归一化）后，与token在value空间中的映射（即矩阵V）相乘，从矩阵乘法的角度看，本质上就是将这种token之间的相似性分数加权聚合到每个token的高维向量中的对应位置。

总之，理解attention的机制，本质就是理解矩阵乘法的含义，后者弄清楚了，前者便是水到渠成的。

2. 为什么除以$\sqrt{d_k}$

我们回到矩阵乘法 $QK^T$中，分别取两个矩阵中的一个向量如下：
$$q = (q_1, q_2, \dots, q_{d_k})^T, \quad k = (k_1, k_2, \dots, k_{d_k})^T$$
考虑二者点积 $s = q \cdot k = \sum_{l=1}^{d_k} q_l k_l$。我们假设q和k的每个分量独立且服从标准高斯分布 $q_l, k_l \sim \mathcal{N}(0,1)$（为什么可以，待会讲），于是就有：
$\mathbb{E}[s] = \mathbb{E}\left[\sum_{l=1}^{d_k} q_l k_l\right] = \sum_{l=1}^{d_k} \mathbb{E}[q_l k_l]$，由于 $\mathbb{E}[q_l k_l] = \mathbb{E}[q_l] \cdot \mathbb{E}[k_l] = 0 \cdot 0 = 0$，于是 $\mathbb{E}[s] = \sum_{l=1}^{d_k} 0 = 0$，即任意两个向量的点积期望为0。

另一方面，我们考虑方差：
$\text{Var}(s) = \mathbb{E}[s^2] - (\mathbb{E}[s])^2 = \mathbb{E}[s^2] - 0 = \mathbb{E}[s^2]$，其中 $s^2 = \left(\sum_{l=1}^{d_k} q_l k_l\right)^2 = \sum_{l=1}^{d_k} (q_l k_l)^2 + \sum_{l \neq m} q_l k_l q_m k_m$。
于是有： $\mathbb{E}[s^2] = \sum_{l=1}^{d_k} \mathbb{E}[(q_l k_l)^2] + \sum_{l \neq m} \mathbb{E}[q_l k_l q_m k_m]$。因为对于任意 $l \neq m$都有 $q_l \perp q_m$， $k_l \perp k_m$，那么可得交叉相期望为0，下面考虑对角项：
$$\mathbb{E}[(q_l k_l)^2] = \mathbb{E}[q_l^2 k_l^2] = \mathbb{E}[q_l^2] \cdot \mathbb{E}[k_l^2] \quad \text{（独立性）}$$
而： $\mathbb{E}[q_l^2] = \text{Var}(q_l) + (\mathbb{E}[q_l])^2 = \sigma_q^2 + 0 = \sigma_q^2$，同理也有 $\mathbb{E}[k_l^2] = \sigma_k^2$，因此 $\mathbb{E}[(q_l k_l)^2] = \sigma_q^2 \cdot \sigma_k^2$。综上所述我们得到：
$\text{Var}(s) = \sum_{l=1}^{d_k} \sigma_q^2 \sigma_k^2 + \sum_{l \neq m} 0 = d_k \cdot \sigma_q^2 \sigma_k^2$，即点积的方差与模型隐藏维度成正比。于是为什么除以 $\sqrt{d_k}$就一目了然了，因为这个放缩操作能将点积的方差变成不依赖维度大小的常数，更稳定，更易于模型训练。

3. 为什么可以假设$q_l, k_l \sim \mathcal{N}(0,1)$？

我们有几种解释的角度：
来源 1：Xavier 初始化

投影矩阵 $W_Q, W_K$ 采用 Xavier（Glorot）初始化：

$$W_{ij} \sim \mathcal{N}\left(0, \frac{2}{d_{\text{in}} + d_{\text{out}}}\right)$$

对于 $W_Q \in \mathbb{R}^{d_{\text{model}} \times d_k}$，每个元素方差约为 $\frac{2}{d_{\text{model}} + d_k} \approx O(1/d_{\text{model}})$。

来源 2：中心极限定理

查询向量分量 $q_l = \sum_{m=1}^{d_{\text{model}}} x_m W_{m,l}^{(Q)}$ 是 $d_{\text{model}}$ 个独立随机变量之和。由中心极限定理，当 $d_{\text{model}} \to \infty$ 时，$q_l$ 收敛于高斯分布。

来源 3：Layer Normalization

Transformer 使用 LayerNorm 将每层输入归一化为零均值、单位方差，进一步保证了 $q_l, k_l$ 的方差接近 1。

综合

$\text{Var}(q_l) \approx 1, \quad \text{Var}(k_l) \approx 1 \quad \Rightarrow \quad \sigma_q = \sigma_k = 1$，这就是 $\sigma_q = \sigma_k = 1$ 假设的严格来源。
